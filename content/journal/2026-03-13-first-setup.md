GitHub Pages 사용자 사이트 저장소를 Quartz 기반의 위키형 블로그 구조로 전환했다.

## 결정한 것

- 작성 경험은 Obsidian 스타일을 우선한다.
- 콘텐츠는 `content/`에서 관리한다.
- 배포는 GitHub Actions로 자동화한다.
